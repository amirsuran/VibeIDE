# Монетизация

## Принцип

VibeIDE — **полностью бесплатный open-source** инструмент.  
**Никаких** ограниченных фич, Pro-плана, таймеров. Все фичи — всем, всегда.

Целевая аудитория — разработчики платящие $20–100/мес за Cursor.  
Часть пересядет, сэкономит и направит часть сэкономленного обратно.

## Трек 1: Донаты и спонсорство (старт сразу)

| Канал | Описание |
|---|---|
| **GitHub Sponsors** | Индивидуальные и корпоративные; кнопка с первого дня |
| **Open Collective** | Прозрачные расходы фонда; соответствует нарративу прозрачности |
| **Корпоративное спонсорство** | Логотип на сайте и в README; без привилегий в фичах |
| **Buy Me a Coffee / Ko-fi** | Минимальный барьер входа для случайных благодарностей |

> Модель работает у: Neovim, Zed, Helix, WezTerm.

---

## Трек 2: Gateway (удобство, не обязательный платёж)

`vibe-gateway` — облачный прокси к моделям.

| Режим | Стоимость | Ограничения |
|---|---|---|
| **Свои API-ключи** | Бесплатно | Нет ограничений, все фичи |
| **Gateway** | Наценка на поддержку проекта | Один счёт вместо ротации ключей |

Gateway — как хостинг у Vercel vs self-host: код тот же, платишь за то чтобы не возиться.

> **Важно:** синхронизация файлов (`.vibe/`) идёт **не через gateway** — через VSCodeSyncFiles в облаке пользователя.

### До запуска gateway (обязательно)
- [ ] ToS
- [ ] Compliance с OpenAI/Anthropic usage policies
- [ ] GDPR для EU (data residency decision — риск #44)
- [ ] Архитектурно отделить от privacy-трека
- [ ] Gateway threat model создан (риск #88)

## Трек 3: Экосистема (долгосрочно)

Не монетизация — flywheel привлечения лучших разработчиков:

- **Community modes marketplace** — авторы популярных modes получают видимость и репутацию
- **MCP Marketplace** — разработчики MCP-серверов получают трафик
- **Contributor recognition** — публичный список контрибьюторов; роль в Discord
