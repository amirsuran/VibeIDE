## [договорённость] Ночной прогон roadmap: rule + skill

**Контекст:** автономное закрытие `docs/roadmap.md` без остановки после одной «тройки»; усиление — максимум пунктов за ночь, без залипания и без лазейки «dev-engine сказал ждать подтверждения».

**Суть:** **`.cursor/rules/roadmap-night-autopilot.mdc`** (`alwaysApply: true`): триггеры в т.ч. `ночной прогон`, `roadmap night`, `пропусти неудачные`, `не получилось иди дальше`, `максимум пунктов роадмапа`, `skip failed roadmap`; главный KPI ответа — **число новых `[x]`** (≥3 порций — нижняя планка, не стоп). Цель — **максимум честных `[x]`**; **ложный массовый defer без содержательной попытки** на каждый пункт — лазейка, **запрещено** (исключение: секрет/human-only из SKILL); **запрещена самоостановка** из-за «мало контекста», «спросить человека» и т.п. (только hard stop из SKILL или обрезка платформы).

Протокол — **`.vibe/skills/roadmap-night/SKILL.md`**: в **одном** ответе цикл **порций** (до 3 верхнеуровневых `- [ ]`, **git commit** после порции; не `git add .`); порядок **сверху вниз**, без перескока к простым задачам ниже. **Не закрывается** пункт после короткого лимита → `> **Roadmap night (deferred):** причина` под строкой, `- [ ]` без `[x]`, в **этом** ответе пункт не повторять; **defer не отменяет** обработку **остальных** пунктов **той же** тройки; при **≥ 2** подряд бесполезных tool-rounds на одном пункте → сразу defer.

Откат diff при defer — по SKILL (в т.ч. entangled batch). Hard stop сужен (секреты → defer; красный compile из пункта → откат пункта + defer + дальше при зелёном базисе). **`CONTINUE_NEXT_MESSAGE`** только при объективной обрезке платформы.

**`dev-engine.mdc`**: явное **исключение roadmap-night** — без ожидания подтверждения после плана. **`roadmap.mdc`**: в night режиме пункты по SKILL in scope; **не блокировать** очередь из-за одного провала — defer и дальше. Финал чата: `[X]` + deferred. Чат: **`/skill:roadmap-night`**.

**Применение:** перед сном один триггер; оператор читает `deferred` в `docs/roadmap.md` и в финале ответа.
