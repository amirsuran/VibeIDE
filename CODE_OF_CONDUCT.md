# Code of Conduct

The VibeIDE community adopts the **Contributor Covenant, version 2.1** as its code of
conduct. The full, canonical text — including scope, expected behavior, enforcement
guidelines, and the four-tier enforcement ladder (Correction → Warning → Temporary Ban →
Permanent Ban) — is published at:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/

A signed text snapshot mirrored at the same URL is available from the Contributor
Covenant project under a Creative Commons Attribution 4.0 license.

## Scope

This code applies to all VibeIDE community spaces — the GitHub repository, issues, pull
requests, discussions, the Discord server (when launched), and any other venue where a
person represents the project.

## Reporting

Reports of unacceptable behavior go to the maintainer privately:

- Email: **i@borodatych.ru** (subject prefix `[VibeIDE CoC]`).
- For the GitHub repository, you may also use the **Report content** option on a comment
  or issue.

The maintainer will acknowledge receipt within a reasonable time, investigate
proportionally, and apply the enforcement ladder as defined in Contributor Covenant 2.1
§ "Enforcement Guidelines". The reporter's identity is kept confidential.

## Maintainer commitments

- The maintainer enforces the code consistently and without favoritism.
- Decisions are documented internally for the community's benefit and to inform any
  appeal.
- Conflicts of interest (e.g. the maintainer is the subject of a report) are escalated to
  another long-term contributor before any action is taken.

## Updates

This file changes only by pull request, with the version number of the adopted
Contributor Covenant kept in sync with the canonical text we link to.
